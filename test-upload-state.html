업로드 상태 지속성 테스트 페이지

이 페이지는 Outline Web Clipper의 업로드 상태 지속성을 테스트하기 위한 페이지입니다.

테스트 시나리오

  1. 이 페이지를 Outline에 저장 (많은 이미지 포함)
  2. 업로드 진행 중 팝업을 닫기
  3. 팝업을 다시 열어서 업로드 진행 상태 확인
  4. 중지 버튼으로 업로드 중단 가능한지 확인

업로드 진행 중:

  • 팝업을 닫고 다시 열어도 진행 상태가 표시되는가?
  • 진행률이 정확하게 표시되는가?
  • 중지 버튼이 표시되는가?
  • 중지 버튼을 누르면 업로드가 중단되는가?
